# 🏥 Senior-Easy App

Aplikacja dla seniorów ułatwiająca codzienne życie i kontakt z rodziną.

## ✨ Funkcjonalności

- 🎥 **Rozmowy wideo** - łatwy kontakt z rodziną
- 💊 **Przypomnienia o lekach** - nigdy nie zapomnij o dawce
- 📞 **Kontakty rodzinne** - szybkie połączenia głosowe i wideo
- 📍 **Lokalne usługi** - dostęp do ważnych numerów (transport, apteka, pomoc domowa)
- 🔔 **Powiadomienia** - ważne informacje w jednym miejscu
- ⚙️ **Tryb opieki rodzinnej** - automatyczne powiadomienia dla rodziny

## 📱 Jak używać

1. Otwórz aplikację w przeglądarce
2. Nawiguj między sekcjami używając dużych, czytelnych przycisków
3. Dodawaj leki z godzinami przypomnienia
4. Kontaktuj się z rodziną jednym kliknięciem

## 🛠️ Technologie

- React 18
- Tailwind CSS
- Vanilla JavaScript
- GitHub Pages
